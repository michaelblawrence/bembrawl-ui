# Bembrawl
A game in progress.

## Contributing

### Hosting

Once running, the mobile app is available at the root (`/`), and the tv app is available at `/tv`.

### Available Scripts

#### `yarn start`

Runs the app in the development mode.<br />
Open [http://localhost:3000](http://localhost:3000) to view it in the browser.

#### `yarn test`

#### `yarn build`
